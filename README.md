PrepIQ - Pure Python Interview Simulator

Description:

PrepIQ is a lightweight, high-efficiency tool designed for 1-on-1 interview practice. It addresses the "Interview Preparation Gap" by providing students with a realistic, distraction-free environment to practice Technical and HR rounds without the need for expensive AI tokens, internet connectivity, or external APIs.

Key Features

Dual-Mode Simulation: Choose between specialized Technical Rounds (Data Structures, OOP, Databases) and HR/Behavioral Rounds.

Zero-Dependency Logic: Built entirely on Pure Python. It is fast, secure, and runs on any machine instantly.

Automated Performance Report: At the end of every session, the system analyzes user responses for "Professional Keywords" and provides a readiness score.

Adaptive Flow: Uses structured loops and time-delayed processing to simulate the pace of a real-world interview.

How to Run (For Judges)
Since this project uses no external libraries (like Flask or OpenAI), the setup is instant.

Open your Terminal or Command Prompt.

Navigate to the project folder.

Run the application using the following command:

Bash
python main.py
Tech Stack
Language: Python 3.x

Architecture: Procedural Logic with Dictionary-based Data Mapping.

Analysis Engine: Keyword-based heuristic scoring.

Why PrepIQ Matters?
In large institutions, one-on-one coaching is often impractical. PrepIQ delivers a standard tool for students to identify and close knowledge gaps before they face high-stakes real-world recruiters.
